# phpbench
## examples

Run specific benchmark class
```shell
make run CLASS_PATH=SubstrVsStringIndexBench
```

```shell
make run CLASS_PATH=StringSplittingBench
```

```shell
make run CLASS_PATH=CheckIfValueIsInArrayBench
```

Run all benchmarks
```shell
make run-all
```
